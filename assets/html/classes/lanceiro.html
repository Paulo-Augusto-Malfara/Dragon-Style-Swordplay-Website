Lanceiro

As Lanças foram utilizadas em praticamente todos os períodos da história humana, alguns estudiosos acreditam que a lança foi uma das primeiras armas que o homem utilizou. Eram muito utilizadas em formações de linha de frente, para segurar o avanço inimigo. Também foram muito utilizadas contra cavalarias, devido ao seu grande alcance.


Dessa forma, não poderíamos deixar de criar a classe Lanceiro, que no Dragon Style é a classe que faz uso das Armas Longas que possuem haste, como Lanças Curtas, Lanças Longas, Alabardas entre outras.

Características

  • É uma classe com um potencial ofensivo muito grande, possui o potencial de atingir inimigos no médio e principalmente no longo alcance nas batalhas corpo-a-corpo, costumam ser muito boas também para batalhas em grupo.
  • Recomendado para pessoas que gostam de manter distância nas batalhas, e que possuem uma boa agilidade para se esquivar de golpes. O Lanceiro não possui uma boa defesa de início, mas ganha acesso a um Broquel em níveis mais altos.

Regras Individuais da Classe

  • O Broquel é um equipamento de defesa exclusivo dos Lanceiros. Pode ser utilizado juntamente com a Lança de duas mãos;
  • Se o Lanceiro quiser, pode lutar de forma parecida com os Hoplômacos, que era uma categoria de Gladiador da época de Roma. Os Hoplomachus lutavam de forma parecida com os Hoplitas, mas utilizando um escudo pequeno, no Dragon Style esse escudo deve respeitar as dimensões máximas do Broquel. A diferença para o Hoplita nesse caso, se dará no tamanho do escudo utilizado;
